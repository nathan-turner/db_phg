Smart PHP Calendar

INSTALLATION

1. Extract smartphpcalendar.zip, copy calendar folder to your webroot folder.
2. Define your MySQL Settings and License Key in calendar/config.php
3. Type http://your-host/smartphpcalendar/install


- You can create administrators and users in administration panel "Users" tab.
- For email reminders cron should be set for each minute with "http://your-host/smartphpcalendar/smart.reminder.php" command
