Install Software
The correct software configuration is required so you can access the XML version of this page.
If any of the 3 checkboxes below are not checked, your system is not configured correctly.
There are 5 Steps.
  • If any of the checkboxes below are NOT checked, you must follow the instructions for that step.
  • Steps 4, and 5 must be done AFTER Steps 1, 2, and 3. For example, if you reinstall Internet Explorer 5.5 (or a later version), you must re-run Steps 2 through 5.
  • Be sure to REBOOT your computer after completing the required steps.
Step 1. Required Browser and version
Microsoft Internet Explorer 5.5 is required. Click below to install.
Step 2. Required Software and version
The Microsoft Parser (Interpreter) "MSXML3" must be installed. Click below to install.
Step 3. Refresh this page
Step 4. Required Software Configuration
The MSXML3 Parser must be set in "Replace" Mode (NOT Side-by-Side mode). Click below to update your settings.
Step 5. Reboot!
